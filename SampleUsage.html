Example usage

    My Quick Links:
      WorkLink A [Using this for A task] - Username/Pass
      WorkLink B [Using this for B task] - Username/Pass

    *Info: Ctrl + S to save note locally

    *Info: Ctrl + Click to open a link in new tab

    *Info: As a part of html, you must type <br> to make a new line for general typing

    *Info: Always include https:// when making a href, E.G.: https://www.google.com

    *Info: Remember to save before refreshing page or redirect from your links. Use the "Re-render" button to rerender your text if needed

    *Info: Feel free to edit your version of the html note file manually and add your own features

    *Info: Source File here: https://github.com/raymondcjyi/NoteHTML

    *Info: If you think there are better features, please contact me or just contribute via GitHub! Thanks!
